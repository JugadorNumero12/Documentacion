Genera una nueva liga

generaLiga(\int[][] $emparejamientos, int $dentro_de, $descanso, int[] $horas) : void

Genera una liga nueva que: - empezará en '$dentro_de' dias, - dejará '$descanso' dias entre jornadas (si metes null escoje el minimo que no superpone jornadas) - sus partidos se jugarán a las '$horas' (si hay pocas horas, se juegan el día anterior en el mismo horario).

Las horas se introducen como un entero entre 0 y 24, que corresponde con la hora GMT+0 Ej: en verano: array(20, 27.5, 13) => los partidos se jugarán a las 22:00, 19:30, 15:00 (hora española) en invierno: array(11.25, 9) => los partidos se jugarán a las 12:15, 10:00 (hora española)

Los partidos se generan en el orden definido en '$emparejamientos'.

WARNING todo el algoritmo esta diseñado para tomar como refernecia de cambio de día las 0:00 GMT

Parameters

$emparejamientos

\int[][]

matriz de equipos emparejados generada en la funcion calendario

$dentro_de

int

dias entre partidos

$descanso

dias de descanso

$horas

int[]

horas a las que se juegan los partidos

Exceptions

\Exception excepcion interna
