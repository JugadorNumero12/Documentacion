    All controller classes for this application should extend from this base class.

    Expulsar a un jugador participante en una accion grupal.

    actionExpulsar($id_accion, $id_jugador) 

    Los recursos que puso el jugador le son devueltos (comprobando limite de animo e influencias)

    route jugadorNum12/acciones/expulsar/{$id_accion}/{$id_jugador}
    redirect jugadorNum12/acciones/ver/{$id_accion}

    Parameters

    $id_accion

    : ide de la accion de donde se va a expulsar al jugador

    $id_jugador

    : ide del jugador que se va a expulsar

    Muestra todas las acciones (individuales y grupales) desbloqueadas Las acciones que el usuario no pueda hacer (por falta de recursos) aparecen remarcadas

    Permite participar en una accion grupal abierta por tu aficion.

    actionParticipar($id_accion) 

    Muestra el formulario que define que recursos va a aportar a la accion que se recogen por $_POST

    El id del jugador se recoge de la variable de sesion

    route jugadorNum12/acciones/participar/{$id}
    redirect jugadorNum12/equipos/ver/{$id_equipo}

    Parameters

    $id_accion

    : id de la accion en la que se va a participar

    Ejecuta la accion (individual o grupal) pulsada (no habrá acciones pasivas o de partido) Significa "bajarse la carta de habilidad"

    actionUsar($id_accion) 

    Cualquier habilidad resta los recursos iniciales al jugador, además,

    Si es una accion grupal muestra un formulario para recoger la cantidad inicial de recursos que aporta el jugador (podría no aportar recursos), Los datos del formulario se recogen por $_POST y se crea una nueva accion grupal en el equipo al que pertenece el usuario Si es una accion individual se ejecuta al momento

    El id del jugador y la aficion a la que pertence se recogen de la variable de sesion

    route jugadorNum12/acciones/usar/{$id_accion}
    redirect jugadorNum12/equipos/ver/{$id_equipo} si es accion grupal
    redirect jugadorNum12/usuarios/perfil si es accion individual

    Parameters

    $id_accion

    : id de la accion que se ejecuta

    Muestra la informacion relativa a una accion grupal abierta - recursos totales requeridos en la accion - jugadores que participan en ella - recursos aportados por cada jugador - efecto si se consigue la accion

    actionVer($id_accion) 

    Si es el usuario que la creo, muestra ademas - botones para expulsar participantes

    route jugadorNum12/acciones/ver/{$id_accion}

    Parameters

    $id_accion

    : id de la accion grupal que se muestra
