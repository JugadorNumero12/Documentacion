Muestra la informacion previa a un partido.

actionPrevia(int $id_partido) : void

1 Si el partido ya se jugo, mostrar la cronica (resultado) de ese partido 2 Si el partido no se ha jugado aún y no es el partido próximo mostrar: - fecha y hora - equipos que jugarán 3 Si el partido es el próximo partido del equipo, mostrar: - fecha y hora - ambiente para el partido - equipo local y visitante - detalles de ambos equipos (aforo previsto, nivel de los equipos) - acciones completadas por las aficiones

route jugadorNum12/partidos/previa/{$id_partido}
redirect jugadorNum12/partidos/index si se intenta asistir a un partido posterior al siguiente

Parameters

$id_partido

int

id del partido sobre el que se consulta la previa
